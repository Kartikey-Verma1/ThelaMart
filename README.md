# Thela Mart

Thela Mart is a web-based platform that connects street vendors with verified raw material providers.  
It allows providers to register and showcase their products while enabling vendors to browse and connect with suppliers easily.

> This project was built primarily for frontend practice and MERN stack learning.

---

## Live Demo

🔗 https://thela-mart.vercel.app

## GitHub Repository

🔗 https://github.com/Kartikey-Verma1/ThelaMart

## Features

- Provider registration system
- Product listing and browsing
- Vendor-provider connection workflow
- FSSAI number field during provider registration
- Responsive user interface
- REST API integration
- MERN stack implementation

## Installation & Setup

### Clone the Repository

```bash
git clone https://github.com/Kartikey-Verma1/ThelaMart.git
```

### Install Packages
```bash
npm install
```

### Run Frontend (Development Mode)
```bash
npm run dev
```

### Run Backend
#### Go to backend
```bash
cd backend
```

#### Run backend
```bash
npm run start
```
